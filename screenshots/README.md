# Screenshots to Add

Place your final deliverable screenshots in this folder.

Recommended filenames:

- `01_lab2_artifacts.png`
- `02_dockerhub_repository.png`
- `03_pulled_container_running.png`
- `04_inference_response.png`

What to capture:

1. Lab 2 GitHub Actions run page showing downloadable artifacts
2. Docker Hub repository page showing the pushed image
3. Terminal or Docker Desktop showing the pulled image/container running
4. Browser or terminal response from the `/predict` endpoint

Note:

These screenshots must be captured manually from the GitHub/Docker UI or your local
machine after the workflow, Docker push, and inference steps complete.
